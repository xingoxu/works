# Canvas2D

- [Canvas Contribution](https://github.com/xingoxu/canvas-contribution)
  一个基于 Canvas 的前端可视化工具，用来生成类似 GitHub Contribution 的活动热力图，支持自定义数据、颜色和文案，适合做个人数据展示或轻量图表嵌入。

# Electron

- [PS4 Wallpaper Helper](http://works.xingoxu.com/ps4helper/)
  一个使用 Electron 开发的桌面工具，帮助用户更方便地整理、浏览和获取适配 PS4 的壁纸资源，提升主机个性化设置的体验。

# React.js

- [日系直邮代购转运费用对比](http://works.xingoxu.com/buy-calc/)
  一个面向日淘和转运场景的费用对比工具，使用 React 构建，帮助用户快速估算不同直邮、代购和转运方案的总成本，降低下单前的决策门槛。
